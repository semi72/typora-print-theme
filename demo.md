# Print Theme

A **print-optimized** theme for minimal paper usage.

## Features

- Compact **9pt typography** saves paper
- Code blocks use tiny **6pt font** for wide diagrams
- Smart **page breaks** keep content together
- No background on code — clean printing

## Quick Example

> "Less paper, same content."

| Feature | Benefit |
|---------|---------|
| Compact fonts | Saves 30% paper |
| No code borders | Cleaner output |
| Widow control | No orphan lines |
