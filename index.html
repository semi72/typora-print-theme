Why Print Theme?

Save Paper

Compact 9pt typography fits more content per page. Print the same document with significantly fewer pages.

Smart Page Breaks

Headings stay with their content. No orphan lines at page boundaries. Tables and code blocks remain intact.

Wide Code Blocks

Tiny 6pt monospace font for code blocks. Perfect for ASCII diagrams, wide tables, and technical documentation.

Clean Output

No background colors or borders on code blocks. Minimal margins maximize usable space. Professional results.

Cross-Platform

Tested on macOS with specific styles for Windows and Linux. Works everywhere Typora runs.

Easy to Customize

CSS variables for colors, spacing, and fonts. Adjust to your preferences in seconds.

See It in Action

Clean, readable typography in the editor
Optimized print layout with DIN A4 support

Available Variants

Theme File Paper Format
Print print.css DIN A4 (210 × 297 mm)
Print Letter print-letter.css US Letter (8.5" × 11")

Installation

1

Download the theme files

Get print.css and/or print-letter.css from the GitHub repository.

2

Copy to Typora themes folder

Place the CSS files in your Typora themes directory:

macOS: ~/Library/Application Support/abnerworks.Typora/themes/
Windows: %APPDATA%\Typora\themes\
Linux: ~/.config/Typora/themes/
3

Restart Typora and select the theme

Open Typora, go to Themes menu, and select Print or Print Letter.
